Это мой учебный проект, предназначенный для управления заказами клиентов условной мебельной фабрики, АРМ для менеджера условной фабрики.

Проект разработан на платформе .NET Framework 4.7.2. на WPF
В нём реализована работа с базой данных через EntityFramework, реализованы все CRUD операции, использовались языки XML и C# (в том числе LINQ), реализована валидация через ValidationRules, работа с изображениями и сторонними библиотеками для создания документов в Word и прочее.
Проект выполняет свои рабочие функции, но он разрабтывался в конце мая 2023-го, и в нём следовало бы реализовать паттерн MVVM, оптимизировать разметку, улучшить дизайн, написать руководство пользователя и исправить несколько мелких недочётов.

Скрипт БД с данными лежит в папке проекта под названием "scriptDB".
